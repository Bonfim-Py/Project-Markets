# Project-Markets
Projeto visa simular um possível cliente/empresário que trabalha na parte econômica de empresas atacadistas. Neste vies, os códigos proporcionam uma limpeza, análise e interpretação do dataframe fornecido por esse soposto cliente. 

## Estruturação

```
-> Data_frame_Market.csv
-> Início.py
-> Perg_1.py
-> Perg_2.py
-> Perg_3.py
-> Perg_4.py
-> Perg_5.py
-> Perg_6.py
-> Perg_7.py
-> README.md
```
## Contextualização 
Foi utilizado o site Kaggle: (https://www.kaggle.com/), para usar supostos datasets. Depois o próprio autor editou e confeccionou a seu modo para que fizesse sentido a análise. Ademais:

```

Início.py = Bibliotecas exportadas
```
